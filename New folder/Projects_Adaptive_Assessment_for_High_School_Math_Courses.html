Adaptive Assessment for High School Math Courses

The first document introduces an adaptive learning system designed to enhance education by tailoring learning experiences for high school students in mathematics. The system predicts students' final scores and selects questions based on their performance, using various tools and libraries like Python, 'numpy,' 'pandas,' and 'scikit-learn.' It employs a Markov chain-based search method for question selection, includes deep learning with the 'MLPRegressor' class, incorporates expert system concepts for transitioning between difficulty levels, utilizes data structures for knowledge representation, and employs symbolic planning to choose questions. The goal is to offer a personalized learning experience, improve efficiency, and enhance student achievement, aligning with the value of adaptive learning systems in education

The Python file that follows depicts the following functionalities to provide personalized assessments and feedback to high school students in mathematics: data loading and preprocessing, a machine learning model, evaluation by calculating both mean squared error and R-squared scores, assessment generation, student assessment, and an output of assessment score, feedback, and a question-wide breakdown of answers.

The last two files consist of assessment questions and student data that includes various features such as the scores of Algebra, Geometry, and Pre-Calculus.
