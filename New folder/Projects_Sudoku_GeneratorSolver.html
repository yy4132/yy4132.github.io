Sudoku Generator/Solver

This project consists of two Python files and one document file. The first Python file is the sudoku generator which consists of functions that generate the appropriate nine by nine matrix, checks whether each number is in valid locations, and solves the generated puzzle. The second Python file consists of two methods of solving the generated Sudoku puzzle through Constraint Programming and Integer Programming. It also compares the performance of both methods over a sample of a random number of Sudoku puzzles. The last file is a report on the algorithms used on solving classic Sudoku.
