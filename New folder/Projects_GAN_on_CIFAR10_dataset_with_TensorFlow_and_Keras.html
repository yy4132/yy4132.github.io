GAN on CIFAR-10 dataset with TensorFlow and Keras

This project consists of two documents and one Python file. The first documents displays the network performance on the generative adversarial network (GAN) along with its parameters and results of first and final epoch. The second document reports on the potential applications and the benefit of generative adversarial networks. The last file implements a simple GAN to generate images similar to a specific class of images from the CIFAR-10 dataset, using TensorFlow and Keras.
